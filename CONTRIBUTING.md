# Contributing

## Policy
Issues and pull requests are welcome. Try to keep changes understandable and update README accordingly.
Please remeber to always be respectuful.

## Developements
Future developements of this projects aims at transforming it in a more general tool to assess compliance to air quality guidelines, along with providing meaningful air quality analyses for citizens, organizations and researchers. <br>

### Goals
- Spliting functions for reusability (expecially in results)
- Expanding configuration options (ex. specifing desired outcomes)
- Improving API calls, improving function for more robust (or faster) calls
- Generalizing functions to be able to handle different timezones
- Improving data cleaning and aggregation for reliability (expecially filtering invalid values)
- Providing additional quality assessments for enhancing understanding of inferences reliability
- Adding further comparison with guidelines (ex. WHO, EPA)

### Structure
- The project package is intended to be a test and a new one will be developed for the general purposes
- If the changes pertains to the project (Italy/EU 2030 focus) should modify only the project folder
- If the contribution pertain to the general tool, please fork or create a new package
